즉, 리스트 = 변수 여러 개를 묶는 역할 

a = [1,2,”int”,”김정현”, [“김재원”,”Manse samdori”]]
-> 리스트 안에 리스트 가능 </br>

## 02-3 리스트 자료형 

- 리스트 인덱싱
</br>
 a = [1,2,3] 일때 a[0]하면 1 반환
a[0] + a[2] 하면 4 반환
a[-1] 하면 3 반환

- 리스트 슬라이싱
</br>
 a = [1,2,3,4,5]
print(a[0:3]) 이라고 하면 0~ 3 미만 까지 이므로 [1, 2, 3] 출력

- 리스트 더하기 
</br>
a = [1,2,3]
b = [4,5,6]
a + b하면 [1,2,3,4,5,6]

- 리스트 반복하기
</br>
a = [1,2,3]
print(a * 3) 하면 [1,2,3,1,2,3,1,2,3] 

- 리스트에서 하나의 값 수정하기
</br>
a = [“박주하", “잠수”, “김은하"]
a[0] = “한재성”
